A full overhaul of Disney's Extreme Skate Adventure.

This is a solo project I'm working on to finally get the PS2 version of Disney's Extreme Skate Adventure looking and sounding as good as it feels to play. The PS2 version has always had the best controls, but the blurry graphics and crunchy audio compared to the Xbox version have always bugged me. I'm fixing that.

**STATUS:** In Development. No public download or patcher yet.

---

### The Plan

* **Hand-Made UI:** I am not using any AI upscaling for this. I'm manually redrawing the menus, HUD icons, and fonts by hand so they actually stay sharp at high resolutions. I want it to look clean and native, not waxy or distorted like those AI filters.
* **Xbox Asset Port:** I'm replacing the compressed PS2 environment textures (ground, walls, skyboxes) with the original uncompressed files from the Xbox version.
* **Audio Upgrade:** The soundtrack is a huge part of this game. I'm swapping out the muffled PS2 audio for the high-quality music and SFX from the Xbox version so it actually sounds clear.
* **Native Widescreen:** I've got engine-level 16:9 support working. The 3D view and HUD are scaled properly so nothing looks stretched or wide.
* **Original Models:** I’m keeping the stock PS2 models to make sure the game stays stable and doesn't crash the engine.

---

### Important Note

**PCSX2 ONLY:** This mod is being built specifically for the emulator to handle the high-res texture replacements and the extra audio memory. **This will not work on an actual PS2 console.**
